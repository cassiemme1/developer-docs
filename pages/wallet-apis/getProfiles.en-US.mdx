# getProfiles

Get Lens Profiles under a wallet address.

## Usage

```js
await connector.getProfiles(address);
```

## Parameters

- `string` - wallet address.

## Returns

```js
Promise<{
    id: string;
}[]>
```

```js
[
  {
    id: '0x0230',
  },
  {
    id: '0x02d4',
  },
];
```
